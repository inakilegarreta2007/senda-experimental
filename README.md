# Senda - Red de Impacto Social

Aplicación para la gestión y visualización de instituciones de ayuda social en la Provincia de Santa Fe.

## Configuración Inicial

1.  **Instalar dependencias:**
    ```bash
    npm install
    ```

2.  **Configurar variables de entorno:**
    Crea un archivo `.env` en la raíz del proyecto y agrega tu API Key de Google (Gemini) y Maps si es necesario.
    ```env
    VITE_API_KEY=tu_api_key_aqui
    ```

3.  **Iniciar servidor de desarrollo:**
    ```bash
    npm run dev
    ```

## Tecnologías
- React + Vite
- TailwindCSS
- Google Generative AI SDK (Gemini)
- React Router DOM
- Leaflet (Mapas)

## Funcionalidades
- **Panel Admin:** Gestión de instituciones y validaciones.

- **Mapa Interactivo:** Visualización de la red de ayuda.
